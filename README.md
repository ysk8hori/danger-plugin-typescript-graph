> Visualize the dependencies between files in the TypeScript code base.

## Usage

Install:

```sh
yarn add danger-plugin-typescript-graph --dev
```

At a glance:

```js
// dangerfile.js
import typescriptGraph from 'danger-plugin-typescript-graph'

typescriptGraph()
```
